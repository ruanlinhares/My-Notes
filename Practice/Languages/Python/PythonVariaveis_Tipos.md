### Variáveis

São espaços nomeados da memória que armazenam algum valor dos tipos acima. 
```
nome = "Harry Potter"

idade = 16

altura = 1.79

trouxa = False
    
```
Em Python diferente de outras linguagens como Java ou C, não é necessário delaclarar o tipo do dado antes da variável.
